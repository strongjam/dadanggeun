# 📅 2개월 개발 로드맵: 다당근 (Global Edition)
> **외국인 거주자를 위한 신뢰 기반 중고거래 플랫폼**

---

## 🚀 개발 전략: "Global-First, Trust-Centered"
1. **i18n Early Adoption**: 다국어 환경(영어/한국어 등)을 1주차부터 기본 아키텍처로 설정.
2. **Translation Bridge**: 판매자와 구매자 사이의 언어 장벽을 실시간 번역 채팅으로 해결.
3. **Safety & Trust**: 외국인 등록증/대학 이메일 인증 및 지도 기반 '세이프존' 추천.
4. **Expat Lifestyle**: 귀국 정리용 '묶음 판매' 및 생활 정보 커뮤니티 강화.

---

## 🏗️ 기술 스택 (Tech Stack)
- **Frontend**: React (i18next 활용 다국어 대응)
- **Backend**: Node.js (Express) - 메인 로직 및 실시간 통신
- **Data/AI**: Python (Google Translate API / DeepL API 연동 및 데이터 분석)
- **DB**: MySQL
- **Realtime**: Socket.io (실시간 번역 채팅 엔진)
- **Maps**: Google Maps API / Kakao Map API (거래 세이프존 표시)

---

## 🟢 1~2주차: 글로벌 기초 세팅 + 다국어 구조 설계
### 🎯 목표: 서비스 골격 및 다국어 환경 완성
- **프로젝트 초기화**
  - i18next를 활용한 **다국어(English/Korean) 대응 구조** 설계.
  - API 검색 인덱싱 설계 (한글/영어 동시 검색 최적화).
- **데이터베이스 모델링 (MySQL)**
  - `users`: 국적, 선호 언어, 인증 상태 필드 추가.
  - `products`: 다국어 제목/설명 대응 및 위경도 정보.
- **기본 UI 컴포넌트**
  - 언어 선택 기능이 포함된 메인/등록/검색 페이지.

---

## 🟡 3~4주차: 핵심 거래 기능 + 신뢰 인증 시스템
### 🎯 목표: 상품 거래의 핵심과 사용자 신뢰 기틀 마련
- **🛍️ 상품 기능 고도화**
  - **Bundle Sale (묶음 판매)**: 이사/귀국 정리 사용자를 위한 묶음 등록 기능.
  - **Multi-Search**: 영문 상품명 입력 시 한글 키워드 자동 매칭 검색.
- **🛡️ 신뢰 인증 (Trust System)**
  - 대학 이메일(`.ac.kr`) 및 거주지 기반 커뮤니티 인증 프로세스.
- **💬 커뮤니티 (Expat Lifestyle)**
  - 비자 정보, 부동산, 생활 팁 등 외국인 전용 Q&A 게시판.

---

## 🔵 5주차: 실시간 번역 채팅 시스템
### 🎯 목표: 언어 장벽 없는 실시간 소통 구현
- **실시간 채팅 (Socket.io)**
  - **In-Chat Translation**: 상대방의 언어로 자동 번역되는 말풍선 기능.
  - 채팅 내 거래 약속 장소 공유 기능.
- **Push Notification**
  - 새로운 메시지 및 거래 제안 알림.

---

## 🔥 6주차: 위치 기반 '세이프존' 추천
### 🎯 목표: 낯선 장소에서의 안전한 직거래 지원
- **현위치 기반 거리 정렬**
  - 사용자 설정 거리별 상품 리스트 필터링.
- **📍 Safe-Trade Zones**
  - 지도 API 연계하여 **경찰서 앞, 지하철 개찰구, CCTV 설치 구역** 등을 거래 추천 장소로 표시.
  - 사용자가 선택한 안전 장소를 채팅방에 바로 공유.

---

## ⚡ 7주차: 스마트 '급처' & 부가 서비스
### 🎯 목표: 거래 회전율 극대화 및 차별화
- **Smart Quick Sale**
  - '급처' 선택 시 자동 가격 할인(예: 30%) 및 **카운트다운 타이머** UI 적용.
  - 메인 상단 'Flash Sale' 섹션 노출.
- **이미지 최적화**
  - AWS S3 업로드 및 다국어 키워드 기반 이미지 태깅.

---

## 🌐 8주차: 다국어 폴리싱 + 안정화 + 배포
### 🎯 목표: 완성도 높은 글로벌 서비스 런칭
- **QA & Localization**
  - 국가별 특수 문자 및 UI 깨짐 현상 최종 점검.
  - 전체 번역 데이터의 정확도 검증 (Native Check).
- **Service Launch**
  - 클라우드 서버 배포 및 운영 모니터링 세팅.

- **No Over-Engineering**: 복잡한 물류 시스템보다는 '정보 연결'에 집중하세요.
- **Legal Compliance**: 개인정보 및 중고거래 관련 법률(전자제품 등) 고지를 영어로 명확히 하세요.

## 🏗️ 최종 MVP 구성
1. 다국어 검색/조회/등록 (KR/EN)
2. **실시간 번역 채팅** (핵심 경쟁력)
3. 위치 기반 **안전 지역 추천** 지도
4. **급처(Quick Sale)** 카운트다운 시스템
5. 커뮤니티 (생활 정보 및 Q&A)
