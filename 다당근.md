## 🟢 1~2주차: 기초 세팅 + 구조 설계

### 🎯 목표
서비스 기본 뼈대 완성

### 📌 작업 내용
- 프로젝트 세팅
  - Frontend: React
  - Backend: Node.js (Express), python
  - DB: MySQL

- 기본 UI 구성
  - 메인 페이지
  - 상품 리스트
  - 상품 등록 페이지

- DB 설계
  - users
  - products
  - chats
  - chat_rooms

---

## 🟡 3~4주차: 중고거래 핵심 기능 + 커뮤니티

### 🎯 목표
상품 거래 + 사용자 참여 기능 강화

### 📌 작업 내용

### 🛍️ 상품 기능
- 상품 등록 (이미지 포함)
- 상품 리스트 조회
- 상품 상세 페이지
- 정렬 기능 (최신순, 가격순)

---

### 💬 커뮤니티 기능 ⭐

#### 1. 게시글 작성
- 자유 게시판 (정보 공유)
- 질문 게시판 (외국인 생활 Q&A)

#### 2. 댓글 기능
- 게시글에 댓글 작성
- 간단한 답변 구조

#### 3. 좋아요 / 추천
- 인기글 노출

#### 4. 위치 기반 게시글 (선택)
- 같은 동네 사람들끼리 소통

---

### 📦 추가 기능
- 이미지 업로드 (AWS S3 또는 로컬 저장)

### 📌 작업 내용

## 🔥 6주차: 위치 기반 기능

### 📍 추가
